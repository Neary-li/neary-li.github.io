Author: Neary Li Date: December 2, 2021  21:42:12 Category: Redis

Redis 安装

1、下载程序包：https://redis.io/

2、利用 xftp 上传至阿里云服务home/admin 目录下

3、命令：mv redis-6.2.6.tar.gz /opt

一般会移动到 opt 目录下

4、进入 opt 目录下 cd /opt

5、解压缩: tar -zxvf redis.xxx.tar.gz

6、进入 redis 解压后的文件 查看配置文件

7、基本的环境安装

使用gcc -v查看是否安装成功

执行make命令:该命令会将所有的环境配置上。

8、redis 的默认安装位置 /usr/loacal/bin

备注：一般我们自己的程序会默认安装至该位置

9、将 reids 配置文件拷贝到当前目录下

以后就使用 myconfig 下的 redis 配置文件，可以保证安全

10、redis 默认不是后台启动的，需要修改配置文件！

11、启动 Redis 服务！

12、使用 redis-cli 进行连接测试！

13、查看 redis 进程是否开启ps -ef|grep redis

14、关闭 redis 服务shutdown
