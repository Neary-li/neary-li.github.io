1、npm 换源

由于 npm 下载源在国外，严重影响速度。因此，乐于分享的淘宝团队，将 npm 下载源部署到了国内。

换源方法

1.1 使用阿里定制的 cnpm 命令行工具代替默认的 npm，输入下面代码进行安装：

npm install -g cnpm --registry=https://registry.npm.taobao.org

以后安装插件只需要使用cnpm intall即可。

1.2 如果习惯了npm，又不想使用cnpm怎么办呢？

输入一下命令：

npm config set registry https://registry.npm.taobao.org

再输入：

npm config list
