MySQL

下载建议：利用压缩包安装，exe 文件安装会存在卸载不干净问题。

1、数据库类型

  • 关系型数据库: (SQL)

    • MySQL ,Oracle, Sql Server, DB2, SQLite
    • 通过表和表之间，行和列之间的关系进行数据的存储

  • 非关系型数据库:(NoSQL) not only

    • Redis, MongoDB
    • 对象存储，通过对象自身的属性来决定

  • DBMS：数据库管理软件

2、安装

  1. 解压
  2. 配置环境变量
    • path 目录下的 bin 路径
  3. 新建 mysql 配置文件*.ini
  1. 启动管理员模式下的 CMD，并切换至 mysql 的 bin 目录下
  2. 输入命令mysqld -install
  3. 启动 MySQL :net start mysql
  4. 再输入mysqld --initialize-insecure --user=mysql初始化数据文件
  5. 然后再次启动 mysql,输入命令mysql -u root -p进入 mysql 管理界面（密码可为空）
  6. 进入界面后更改 root 密码

update mysql.user set authentication_string=password('123456') where user='root' and Host='localhost';

  1. 最后输入flush privileges;刷新权限
  2. 修改 my.ini 文件 注释最后一句 ski-grant-tables
  3. 重启 mysql 即可正常使用

net stop mysql

net start mysql

  1. 链接测试mysql -u root -p123456

3、字段属性

  1. Unsigned:
    • 无符号整数
    • 声明该列不能为负数
  2. zerofill:
    • 0 填充的
    • 不足的位数，用 0 来填充
  3. 自增：
    • 自动再上一条记录的基础上+1
    • 通常用来设计唯一的主键
  4. 非空
  5. 默认

扩展：阿里巴巴规范必须含有的列

显示表的结构DESC student

4、数据表的类型

MYISAM INNODB
事物支持 N Y
数据行锁定 N Y
外键约束 N Y
全文索引 Y N
表空间大小 较小 较大，约为 2 倍

常规使用操作：

  • MYISAM 节约空间，速度较快
  • INNODB 安全性高，事务的处理，多表多用户操作
