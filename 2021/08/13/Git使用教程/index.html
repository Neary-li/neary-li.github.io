Git使用教程

Author: Neary Li Date: August 13, 2021  14:42:03 Category: Git

1、新建

  • git clone [url]: 克隆一个项目和它整个代码历史

  • git init [project-name]: 新建一个目录，并初始化为本地版本库

2、配置

  • git config --list 显示当前的 git 配置

  • git config -e [--global] 编辑 Git 配置文件

设置提交代码时的用户信息

  • git config [--global] user.name “[name]”
  • Git config [--global] user.email “[email address]”

3、增/删文件

  • git add .添加当前目录的所有文件到暂存区

  • Git add [file] 添加指定文件或者目录

  • git rm [file1]删除工作区文件，并将这次删除放入暂存区

4、代码提交

  • Git commit -m [message] 提交暂存区到仓库区

5、分支

  • Git branch 列出所有本地分支

  • Git branch -r 列出所有远程分支

  • Git branch -a 列出所有本地分支和远程分支

  • Git merge 合并指定分支到当前分支

  • Git checkout 切换分支

6、远程同步

  • Git fetch [remote] 下载远程仓库的所有变动
  • Git remote -v 显示所有远程仓库
  • Git push上传本地指定分支到远程仓库
  • Git push [remote] --force 强行推送当前分支到远程仓库，即使有冲突
  • Git push [remote] --all 推送所有分支到远程仓库
  • Git pull [remote] [branch]取回远程仓库的变化，并于本地分支合并

7、查看

  • Git log 显示当前分支的版本历史
  • Git diff 显示暂存区与工作区的差异
  • Git reflog 显示当前分支的最近几次提交

8、撤回回退

  • Git reset
