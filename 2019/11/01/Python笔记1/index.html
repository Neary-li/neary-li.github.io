变量

语法：变量名 = 值

标识符(Python各类名字统一规范)

  • 由数字、字母、下划线组成
  • 不能以数字开头
  • 不能使用内置关键字
  • 严格区分大小写

命名习惯

  • 见名知意
  • 大驼峰：MyName
  • 小驼峰：myName
  • 下划线：my_name

  • 打断点

数据类型

  • 数值型
    • int:a = 1
    • float:a = 1.1
  • 布尔型
    • True:a = True
    • False:b = False
  • str(字符串):a = 'hello'
  • list(列表):b = [1,2,3]
  • tople(元组)：c = (1,2,3)
  • set(集合)：c = {1,2,3}
  • dict(字典)

查看数据类型print(type(变量名))

输出

格式化输出

格式符号 转换
%s 字符串
%d 有符号的十进制整数
%f 浮点数

使用%s进行输出print('我叫%s,今年%s岁' %(name,age))与使用f'{表达式}'进行输出print(f'我叫{name},今年{age}岁')效果一样，后一种为Python3.6中新增写法。

转义字符

  • \n:换行

  • \t:制表符，一个tab键等于4个空格距离

输入

语法:input("提示信息")

输入的特点

  • 当程序执行到input，等待用户输入，输入完成之后才继续向下执行。
  • 在Python中，input接收用户输入后，一般存储到变量，方便使用。
  • 在Python中，input会把接收到的任意用户输入的数据都当做字符串处理。

示例：password = input("请输入密码")

转换数据类型的函数

函数 说明
int(x) 将x转换成一个整数
float(x) 将x转换成一个浮点数
str(x) 将对象x转换为字符串
eval(str) 用来计算在字符串中的有效Python表达式，并返回一个对象
tuple(s) 将序列s转换为一个元组
list(s) 将序列s转换为一个列表

复合赋值运算符

运算符 描述 实例
+= 加法赋值运算符 c += a 等价于c = c + a
-= 减法赋值运算符 c -= a 等价于c = c - a
*= 乘法赋值运算符 c *= a 等价于c = c * a
/= 除法赋值运算符 c /= a 等价于c = c / a
//= 整除赋值运算符 c //= a 等价于c = c // a
%= 取余赋值运算符 c %= a 等价于c = c % a
**= 幂赋值运算符 c **= a 等价于c = c ** a

PS:遇到下面这种情况时先算加法再算复合运算符
